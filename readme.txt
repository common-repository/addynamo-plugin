== Description ==

This plugin inserts the embedding code of your Addynamo ads before, after or in the middle of a post without modify the theme.

The plugin is used with your Addynamo adverts that injects the code into your site without editing the template.

Your ads can be injected into your homepage, post pages, category pages and general pages. You can also choose if you want to display the ads before or after these sections.

Make sure to check out our sidebar widget that displays ads in your sidebar

Read more on the plugin page at http://www.addynamo.com.

== Installation ==

1. Put the plugin folder into [wordpress_dir]/wp-content/plugins/
2. Go into the WordPress admin interface and activate the plugin
3. Go to the settings page and configure the plugin - Addynamo
4. Make sure you are don't use the the_excerpt() tag as the plugin will not work on those pages
